### Push Notifications

<Accordion title="Stay Informed Anywhere" icon="bell">
**Event Types:**
- **Earnings:** Before, during, and after announcements
- **Price Movements:** Significant gains/losses (5%+, customizable)
- **Analyst Ratings:** Upgrades, downgrades, price target changes
- **Insider Trading:** Unusual buying or selling activity
- **News:** Breaking news for your watchlist stocks
- **Dividends:** Declaration, ex-div date, payment reminders

**Customization:**
- Set per-watchlist notification preferences
- Quiet hours (e.g., 10 PM - 7 AM)
- Weekend notifications on/off
- Threshold settings (e.g., only notify for >10% moves)

**Setup:**
1. Open Bloom app
2. Go to Settings ‚Üí Notifications
3. Enable notification types
4. Customize thresholds and quiet hours
5. Grant system notification permissions
</Accordion>

## Tips for Mobile Use

<AccordionGroup>
  <Accordion title="Enable Biometric Authentication" icon="fingerprint">
    **Setup:**
    1. Open Bloom app
    2. Settings ‚Üí Security
    3. Enable Face ID (iOS) or Fingerprint (Android)

    **Benefits:**
    - Quick access to your portfolio
    - Secure your financial data
    - No need to remember password
  </Accordion>

  <Accordion title="Set Up Widgets" icon="grid">
    **iOS:**
    1. Long-press home screen
    2. Tap "+" in top-left
    3. Search "Bloom"
    4. Choose widget size (small, medium, large)
    5. Add to home screen

    **Android:**
    1. Long-press home screen
    2. Tap "Widgets"
    3. Find "Bloom"
    4. Drag to home screen

    **Widget Types:**
    - Portfolio Summary (value, % change)
    - Top Movers (biggest gainers/losers today)
    - Watchlist Overview (quick glance)
  </Accordion>

  <Accordion title="Use Voice Input for AI Questions" icon="microphone">
    **How to Use:**
    1. Open AI Assistant
    2. Tap microphone icon in chat input
    3. Speak your question naturally
    4. Review transcription, edit if needed
    5. Send

    **Example Voice Queries:**
    - "Is Apple a good buy right now?"
    - "What are the top momentum stocks in tech?"
    - "Analyze my portfolio for risks"

    **Tips:**
    - Speak clearly and pause between phrases
    - Review transcription before sending
    - Works best in quiet environments
  </Accordion>

  <Accordion title="Share Stock Insights" icon="share">
    **Share Options:**
    - Copy stock data to clipboard
    - Share screenshot of stock page
    - Export AI conversation as text
    - Send to Messages, Email, Slack, etc.

    **Use Cases:**
    - Share research with financial advisor
    - Discuss stocks with friends
    - Save insights for later
    - Archive investment theses
  </Accordion>

  <Accordion title="Manage Notifications Wisely" icon="bell">
    **Start Conservative:**
    - Enable: Earnings announcements
    - Enable: Major analyst upgrades/downgrades
    - Enable: Price movements >10%
    - Disable: Everything else initially

    **Adjust Based on Usage:**
    - Too many notifications? Raise thresholds
    - Missing important events? Lower thresholds
    - Getting woken up? Set quiet hours

    **Best Practice:**
    - Core holdings: All notifications
    - Research ideas: Minimal notifications
    - Speculative: Major events only
  </Accordion>
</AccordionGroup>

## Technical Details

**Built With:**
- **Framework:** Capacitor v6 (native bridge)
- **Frontend:** React 17 + TypeScript
- **UI:** Chakra UI (mobile-optimized)
- **State:** React Query
- **Build:** Vite

**Performance:**
- First load: < 2 seconds
- Page transitions: < 200ms
- Scrolling: 60 FPS
- Memory usage: < 100 MB

**Security:**
- All data encrypted in transit (TLS 1.3)
- Biometric authentication (device keychain)
- Auto-lock after inactivity
- No sensitive data stored on device

**Data Usage:**
- Average per session: 2-5 MB
- AI chat: ~500 KB per query
- Stock page load: ~200 KB
- Charts: ~100 KB per timeframe
- Offline mode: 0 KB after initial cache

## Troubleshooting

<AccordionGroup>
  <Accordion title="App won't open / crashes on launch" icon="circle-exclamation">
    **Solutions:**
    1. Force quit the app
    2. Restart your device
    3. Check for app updates in App Store / Play Store
    4. Reinstall the app (data is synced, you won't lose anything)
    5. Contact support if issue persists
  </Accordion>

  <Accordion title="Notifications not working" icon="bell-slash">
    **Check These Settings:**

    **In Bloom App:**
    - Settings ‚Üí Notifications ‚Üí Ensure enabled

    **On iPhone:**
    - Settings ‚Üí Bloom ‚Üí Notifications ‚Üí Allow Notifications

    **On Android:**
    - Settings ‚Üí Apps ‚Üí Bloom ‚Üí Notifications ‚Üí Allow

    **Still Not Working?**
    - Try logging out and back in
    - Reinstall app
    - Contact support
  </Accordion>

  <Accordion title="Prices not updating" icon="clock">
    **Possible Causes:**
    - Market closed (updates resume during market hours)
    - Poor internet connection
    - App needs refresh

    **Solutions:**
    - Pull down on watchlist to refresh
    - Check internet connection
    - Force quit and reopen app
  </Accordion>

  <Accordion title="AI Assistant not responding" icon="robot">
    **Check:**
    - Internet connection (AI requires online access)
    - Bloom servers status (check status.investwithbloom.com)
    - Try refreshing the app

    **If Slow:**
    - High usage times (market open) can cause delays
    - Usually resolves in 10-30 seconds
  </Accordion>
</AccordionGroup>
