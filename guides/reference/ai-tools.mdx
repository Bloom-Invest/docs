## Overview

Bloom's AI Assistant has access to 28 specialized financial tools. This reference guide documents each tool, its parameters, and example use cases.

## Core Research Tools (3)

### get_symbols_metadata

**Purpose:** Get comprehensive stock metadata and fundamentals

**Parameters:**
- `symbols`: List of stock tickers (e.g., ["AAPL", "GOOGL"])

**Returns:**
- Current price and market cap
- P/E, P/S, P/B ratios
- Revenue and earnings growth
- Dividend yield
- 52-week high/low
- AI investment rating

**Example Use:**
```
"Get metadata for AAPL, GOOGL, and MSFT"
```

---

### research_stock

**Purpose:** Fundamental analysis with investment ratings

**Parameters:**
- `symbol`: Stock ticker
- `analysis_type`: "quick" or "deep"

**Returns:**
- Investment rating (Buy/Hold/Sell)
- Valuation assessment
- Growth prospects
- Risk factors
- Competitive positioning

**Example Use:**
```
"Research Microsoft in depth"
```

---

### analyze_earnings

**Purpose:** Earnings analysis with estimates and trends

**Parameters:**
- `symbol`: Stock ticker
- `periods`: Number of quarters to analyze (default: 4)

**Returns:**
- EPS actual vs estimate
- Revenue actual vs estimate
- Earnings surprises
- Growth trends
- Forward guidance

**Example Use:**
```
"Analyze Apple's last 4 quarters of earnings"
```

## Advanced Research (3)

### analyze_technicals

**Purpose:** Technical indicators and chart patterns

**Parameters:**
- `symbol`: Stock ticker
- `timeframe`: "1D", "1W", "1M", "3M", "1Y"

**Returns:**
- RSI (Relative Strength Index)
- Moving averages (50/200 SMA)
- Bollinger Bands
- MACD
- Support/resistance levels
- Volume analysis

**Example Use:**
```
"Show me NVDA's technicals on the 1-month chart"
```

---

### search_transcript

**Purpose:** Keyword search in earnings call transcripts

**Parameters:**
- `symbol`: Stock ticker
- `keywords`: List of search terms
- `limit`: Number of results (default: 10)

**Returns:**
- Transcript excerpts matching keywords
- Speaker (CEO, CFO, analyst)
- Context around mentions
- Transcript date

**Example Use:**
```
"Search Tesla transcripts for 'Full Self-Driving' and 'Robotaxi'"
```

---

### get_symbol_charts

**Purpose:** Interactive price charts across timeframes

**Parameters:**
- `symbol`: Stock ticker
- `timeframes`: List of timeframes

**Returns:**
- Chart data (OHLCV)
- Technical overlay data
- Chart image URLs

**Example Use:**
```
"Show me AMD's 3-month and 1-year charts"
```

## Market & Sentiment (2)

### get_market_sentiment

**Purpose:** Unified sentiment analysis from multiple sources

**Parameters:**
- `symbol`: Stock ticker

**Returns:**
- News sentiment score (-1 to +1)
- Social media sentiment
- Analyst rating consensus
- Insider trading sentiment
- Overall sentiment rating

**Example Use:**
```
"What's the market sentiment on Tesla?"
```

---

### fetch_url_content

**Purpose:** Fetch and analyze web content with AI

**Parameters:**
- `url`: Web page URL
- `analysis_prompt`: What to extract/analyze

**Returns:**
- Extracted content
- AI analysis based on prompt
- Key takeaways

**Example Use:**
```
"Fetch and summarize this analyst report: [URL]"
```

## Portfolio Tools (3)

### analyze_portfolio

**Purpose:** Portfolio review with recommendations

**Parameters:**
- `holdings`: Dict of symbol: allocation_percent pairs
- `investor_profile`: "conservative", "moderate", "aggressive"

**Returns:**
- Sector breakdown
- Correlation analysis
- Risk metrics (beta, volatility)
- Diversification score
- Rebalancing recommendations

**Example Use:**
```
"Analyze my portfolio: 30% AAPL, 25% GOOGL, 20% MSFT, 15% NVDA, 10% AMZN"
```

---

### filter_stocks

**Purpose:** Screen stocks by fundamental criteria

**Parameters:**
- `expressions`: List of filter expressions
- `limit`: Max results (default: 50)

**Returns:**
- Matching stock symbols
- Key metrics for each
- Ranking by criteria

**Filter Expression Syntax:**
```
"P/E < 20"
"revenue growth > 20%"
"sector = Technology"
"market cap > 1B"
"dividend yield > 3%"
```

**Example Use:**
```
"Find stocks with P/E < 20, revenue growth > 20%, and debt-to-equity < 1"
```

---

### analyze_trades

**Purpose:** Trade quality analysis for buy/sell decisions

**Parameters:**
- `symbol`: Stock ticker
- `action`: "buy" or "sell"
- `price`: Target entry/exit price

**Returns:**
- Risk/reward assessment
- Entry/exit timing analysis
- Position sizing recommendation
- Stop-loss levels

**Example Use:**
```
"Should I buy TSLA at $250? Analyze the trade quality"
```

**Parameters:**

  - "dividend_aristocrats"
  - "sector_leaders"
  - "ai_innovation"
  - "defensive"

**Returns:**
- List of stocks in collection
- AI reasoning for each
- Key metrics
- Risk factors

**Example Use:**
```
"Show me the Top Momentum Stocks collection"
```

---

### get_all_research_reports

**Purpose:** Recent AI-generated research reports

**Parameters:**
- `limit`: Number of reports (default: 10)
- `filter_by`: "sector" or "theme"

**Returns:**
- Research report titles
- Summary of each report
- Stocks covered
- Publish date

**Example Use:**
```
"Show me the latest AI research reports"
```

## Using Tools Effectively

<AccordionGroup>
  <Accordion title="You Don't Call Tools Directly" icon="info-circle">
    The AI automatically selects which tools to use based on your question.

    **Good:** "Should I buy Apple?"

    **Bad:** "Use get_symbols_metadata to get Apple data" (too explicit)
  </Accordion>

  <Accordion title="Multiple Tools Per Query" icon="layer-group">
    The AI can chain multiple tools:

    **Question:** "Should I buy Tesla at current prices?"

    **AI Uses:**
    1. `get_symbols_metadata` (current price, fundamentals)
    2. `analyze_earnings` (recent results)
    3. `analyze_technicals` (entry timing)
    4. `get_market_sentiment` (sentiment check)
    5. Synthesizes â†’ Recommendation
  </Accordion>

  <Accordion title="Context is Remembered" icon="brain">
    The AI remembers your conversation:

    1. **You:** "Analyze Apple"
    2. **AI:** [Uses tools, provides analysis]
    3. **You:** "What are the risks?" *(knows you mean Apple)*
    4. **AI:** [Focuses on Apple risks]
  </Accordion>
</AccordionGroup>

## Tool Limitations

<Warning>
**Rate Limits:**
- Free tier: 50 AI queries/month
- Premium: Unlimited queries
- Heavy usage may experience throttling
</Warning>

<Warning>
**Data Freshness:**
- Stock prices: 1-minute delay during market hours
- Fundamentals: Updated daily after market close
- Earnings: Real-time (as announced)
- News: Real-time
</Warning>

<Warning>
**Coverage:**
- US stocks: Full coverage (NYSE, NASDAQ, OTC)
- International: Limited (major exchanges only)
- Crypto: Not supported
- Commodities: Not supported
</Warning>
